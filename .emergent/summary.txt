<product_requirements>
Big Mann Entertainment (BME) is a media distribution platform aiming to distribute content across 91+ platforms, incorporating admin controls and industry compliance (DDEX, IPI, ISNI, AARC, MDX, MLC, GS1). It manages UPC, GLN, and ISRC identifiers, with an advanced licensing system.

**Initial MVP Focus:**
*   Stabilize application: Remove WebAuthn, fix password reset, implement email notifications, resolve numerous backend (404s, API endpoints, UPC/ISRC validation) and frontend (JS errors, file upload, distribution display, pricing buttons, 404 page, mobile navigation) issues.
*   Goal: Achieve 100% functional stability and a complete upload-to-library-to-distribution workflow with real-time status.

**Subsequent Enhancements (Implemented Phases):**
*   **AWS Integration:** Integrate AWS S3, SES, CloudFront, Lambda, and Rekognition for scalable media storage, delivery, processing, and moderation.
*   **Enhanced Upload:** Implement a drag-and-drop upload feature for media, artwork, and metadata, with comprehensive metadata entry (title, ISRC, UPC, rights holders) and real-time validation.
*   **Phase 1: Metadata Parser & Validator:** Add support for DDEX ERN (XML), MEAD, JSON, CSV formats with schema validation, required field checks (ISRC, release date, rights), and platform-wide duplicate detection (ISRC/UPC).
*   **Phase 2: Advanced Metadata Features:** Implement batch processing, advanced reporting and analytics, and support for additional metadata formats (ID3, MusicBrainz, iTunes, ISNI).
*   **Phase 3: Rights & Compliance Checker:** Add checks for territory rights (e.g., US, EU, Global), usage rights (streaming, sync, broadcast), and expiry dates with embargo logic.
*   **Phase 4: Smart Contract Trigger:** Auto-trigger licensing or DAO votes on valid content submission using Web3/blockchain integration.
</product_requirements>
