The BME project aims to enhance the Unified Label Network (ULN) and introduce a comprehensive Creator Profile System. The ULN system has implemented admin login, generic label editing, legal identifier addition (IPN 10959387), label type updates, comprehensive label profile updates, and integrated ESPN as a new distribution platform.

The Creator Profile System is currently underway. It employs a dual-database architecture, utilizing PostgreSQL alongside existing MongoDB. The system is designed to support and display profiles for all users, integrate full GS1 metadata (GTIN, GLN, ISRC, ISAN, digital links, QR codes), implement a complete DAO governance system (proposals, weighted voting, automated royalty adjustments), and include OAuth-based social media integration (e.g., Facebook, TikTok). Seamless integration with existing authentication, ULN, media upload/distribution, and earnings dashboards is also a key requirement. The current implementation includes a working backend and a structurally built frontend, with a graceful fallback for external PostgreSQL connectivity issues.
